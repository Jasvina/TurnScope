# TurnScope Community Code of Conduct

TurnScope is an open project for people building, studying, and debugging coding agents.
We want the repository, issues, and review process to feel rigorous, welcoming, and calm.

## Our standards

Examples of behavior that help this project:

- being respectful in disagreement
- focusing critique on ideas, evidence, and implementation details
- sharing context, traces, and reproduction steps instead of assumptions
- making room for contributors with different experience levels
- taking responsibility when we cause harm or confusion

Examples of unacceptable behavior:

- harassment, threats, or discriminatory language
- personal attacks or repeated bad-faith arguments
- publishing private information without permission
- trolling, dogpiling, or deliberately derailing collaboration
- sexualized language or imagery in project spaces

## Scope

This code of conduct applies in repository discussions, issues, pull requests, and any
other public spaces directly connected to TurnScope project work.

## Reporting

If you experience or witness behavior that violates this code of conduct, follow the
reporting path in `SECURITY.md` and include links or screenshots when possible.

## Enforcement

Maintainers may remove comments, close threads, reject contributions, or temporarily or
permanently restrict participation when behavior harms the project or its contributors.
